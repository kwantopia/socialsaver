    Facebook Profile Information

    This page shows all the available tags in the facebook template_tags file

    Tag Name Facebook Value Description
    show_facebook_full_name {% show_facebook_full_name fbuser %} shows the users full name without a link
    show_facebook_name {% show_facebook_name fbuser %} Shows the users full name with link back to facebook profile
    show_facebook_first_name {% show_facebook_first_name fbuser %} show the users first name with link to facebook profile
    show_facebook_last_name {% show_facebook_last_name fbuser %} show the users last name with link to facebook profile
    show_facebook_possesive {% show_facebook_possesive fbuser %} shows the facebook 'possessive' value for the user
    show_facebook_greeting {% show_facebook_greeting fbuser %} A greeting you want to show the user. You can change the greeting in the tag
    show_facebook_photo {% show_facebook_photo fbuser %} shows the cropped users facebook profile image
    show_connect_button {% show_connect_button %} show the facebook connect button
    show_logout {% show_logout %} Show the logout link
    show_invite_link {% show_invite_link %} shows a 'invite your friends' popup that retrieves the list of all your friends
    modify the facebook/invitiation.fbml file to modify look/feel.
    show_facebook_info {% show_facebook_info fbuser %} show some of the users facebook information. Modify the facebook/display.html file to modify the look and feel.
    show_profile_mosaic {% show_profile_mosaic friendList %} show a mosaic of your friends list. The number of friends returned is limited by the settings.NUM_FRIEND_RETRIEVE_LIMIT setting.
